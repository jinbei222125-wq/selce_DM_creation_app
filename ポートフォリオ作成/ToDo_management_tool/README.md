# ToDo管理ツール

シンプルで使いやすいタスク管理Webアプリケーションです。

## 機能

### 基本機能
- ✅ タスクの追加・削除・完了/未完了の切り替え
- 📊 リアルタイム統計表示（総タスク数、完了済み、未完了）
- 🔍 フィルタリング機能（すべて、未完了、完了済み）
- 💾 ローカルストレージでのデータ永続化
- 📱 レスポンシブデザイン（モバイル対応）

### 追加機能
- 🎨 モダンで美しいUIデザイン
- 🔔 通知システム（成功、エラー、情報メッセージ）
- ⌨️ キーボードショートカット
- 🎭 スムーズなアニメーション
- 🗑️ 一括削除機能（完了済み、すべて）

## 使用方法

### インストール
1. プロジェクトをダウンロードまたはクローン
2. `index.html`をブラウザで開く

### 基本的な操作
1. **タスク追加**: 入力フィールドにタスクを入力して「+」ボタンをクリック、またはEnterキーを押す
2. **タスク完了**: チェックボックスをクリック
3. **タスク削除**: 各タスクの削除ボタン（ゴミ箱アイコン）をクリック
4. **フィルタリング**: 上部のフィルターボタンでタスクを絞り込み

### キーボードショートカット
- `Enter`: タスクを追加
- `Ctrl/Cmd + Enter`: タスクを追加
- `Escape`: フィルターを「すべて」にリセット

## カスタマイズ

### 色の変更
`styles.css`の以下の変数を変更することで、アプリケーションの色をカスタマイズできます：

```css
/* メインカラー */
background: linear-gradient(135deg, #667eea 0%, #764ba2 100%);

/* アクセントカラー */
color: #667eea;
```

### 機能の追加
`script.js`の`TodoApp`クラスに新しいメソッドを追加することで、機能を拡張できます。
